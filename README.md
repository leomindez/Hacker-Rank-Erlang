# Hacker-Rank-Erlang
Solutions for Hacker-Rank problems by using Erlang

## Number to Binary 
``` 
    number:number_to_binary(2). 
    [0,1]
```

## 1's Binary number consecutives
```
consecutives:find_consecutives([1,1,0,1].
	2
```

## Left Rotation
```
rotate:left_rotation(4,[1,2,3,4,5]).
	[5,1,2,3,4]
```
